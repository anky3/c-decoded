Operators

An operator can be defined as a special symbol in a language that identifies the operation that can be performed on the operands in the language.

The operators in C are categorized into eight types

C language provides different operators to perform arithmetic operations.

Syntax:

Assignment Operators

The special symbol = is known as an assignment operator in C language. It is used to assign a result of an expression or constant value to a variable.

Example:

			/* an expression assigned to variable area */
			area = (base * height)/2;
			
			/* a constant value 14 is assigned to variable roll_no */
			roll_no = 14;

The special symbol = is also known as a shorthand operator. it is used to reduce the size of expression and improve the readability of C programs. To convert the special operator = into a shorthand operand, we prefix any arithmetic operator to =.

Syntax:

			identifier Arithmetic_Operator= expression;
							/* OR */
			identifier Arithmetic_Operator= constant_value;

Example:

	
			your_num += 5; // equivalent to your_num = your_num + 5;
			my_num *= 2;  // equivalent to my_num = my_num + 2;
			your_num -= 2;  // equivalent to your_num = your_num - 2;
			my_num /= 2;  // equivalent to my_num = my_num / 2;
			your_num %= 2;  // equivalent to my_num = my_num % 2;

Increment and Decrement Operators

C language provides two unary operators, namely, ++ and --, for the purpose of increment and decrement of value by 1. The expression formed by using increment or decrement operator is termed as unary expression, as these operators require only one operand.

Syntax:

		++identifier;		//Pre-increment
		identifier++;		//Post-increment
		
		--identifier;		//Pre-decrement
		identifier--;		//Post-decrement

Example:

			#include<stdio.h>
			#include<conio.h>
			void main()
			{
				int a=6,b;
				clrscr();
				b = --a + --a;
				printf("a = %d\tb = %d\n",a,b);
				b = a-- + a--;
				printf("a = %d\tb = %d\n",a,b);
				b = ++a + ++a;
				printf("a = %d\tb = %d\n",a,b);
				b = a++ + a++;
				printf("a = %d\tb = %d",a,b);
				getch();
			}

Output:

a = 4 b = 8
a = 2 b = 8
a = 4 b = 8
a = 6 b = 8


Conditional Operators

C language provides us an operator known as conditional operator ternary operator for condition checking. It consists of combination of two symbols, that is ? and :.

Syntax:

			(condition) ? (true statement) : (false statement);
