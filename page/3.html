Data Type

The type of data value that can be stored in an identifier is known as its data type.


Primitive

The primitive data types form the basic building block for defining an identifier. C language supports four primitive data types, namely, integer, real, character and empty.


Integer

Many numeric operations involve a positive or a negative integer number. To store integer numbers in an identifier, we need to define the integer of type int. In this case, int is a reserved word, which indicates that the identifier is capable of storing an integer data value.


Real

Most of the financial and mathematical calculations require a real number in calculation. To use real values in C language, we need to use keyword float as the data type.


Character

To store Character data values, we have to use a keyword char.


Empty Data Set

The last one of the primitive data types in C language is void. A void data type doesn’t associate an identifier with a specific type, such as integer or real. It associates identifier with an empty value. It is mainly used along with functions and pointers.


User-defined Data Type

New data type created by the user from the existing data type is called user-defined Data type. Keyword typedef is used to create a new data type.

Syntax: 

			typedef datatype identifier;

Example: 

			typedef int num;
