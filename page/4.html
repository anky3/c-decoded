Input and Output functions

To provide input and output, C language provides inbuilt functions. The functions for input and output are based on the data types and are categorized as formatted functions and unformatted functions.

Formatted Functions

Formatted Functions

The formatted functions are considered to be all-purpose functions. These functions are used to input or output all type of data.


Input Functions

C language provides a pre-defined function called scanf() to take the input

Syntax:

			scanf("format string", address list of identifier);

Output Function

C language provides a pre-defined function called printf() to show the output.

Syntax:

		printf("format specifier", list of identifier);
					/*	OR	*/
		printf("message string");

Unformatted Functions

The unformatted functions cannot format data and work with only a character data type. It do not need a conversion symbol similar to the formatted functions.


Unformatted single character I/O functions

C language provides us with two unformatted functions getchar() and getc() that work on character data type and two output functions putchar() and putc() to give us the output.

Syntax:

			/*	Input	*/
		identifier name = getchar();
		identifier name = getc(stdin);
		
			/*	Output */
		putchar(identifier name);
		putchar (identifier name, stdout);

Unformatted String I/O functions

C language provides us with two unformatted functions gets() and puts() to input and output a string. A string in C is represented as asset of characters that is always terminated with a null character.

Syntax:

		Gets(identifier name);
			/*	AND	*/
		puts(identifier name);
